This file describes how is the Houdini Engine comiled file linked into Maya for developers.

'D:\SCS_PROJECT\art_assets\houdini\run_maya_beta_dev.exe'
    - Alena Mikushina magic that sets the environment 
    - primarily creates the file 'D:\SCS_PROJECT\maya_beta\modules\houdiniEngine-maya2022' that has a reference to the compile_definitions    
        Houdini Engine that should be loaded when Maya is started

'D:\SCS_PROJECT\maya_beta\modules\houdiniEngine-maya2022'
    - the first line '+ houdiniEngine 1.0 ...path...' specifies the path to the compiled HoudiniEngine.mll file 

'D:\SCS_PROJECT\maya_beta\runMayaBeta.exe'
    - run this to load maya with the compiled plug in
